# PulseCRM Mobile

Este diretorio reserva a futura aplicacao mobile nativa.

## Escopo recomendado

- login e sessao segura
- dashboard resumido
- leads e pipeline
- agenda e tarefas
- inbox com notificacoes push

## Stack sugerida

- Expo
- React Native
- TypeScript
- React Navigation
- TanStack Query
- Expo Notifications

## Observacao

No ambiente atual nao ha gerenciador de pacotes disponivel, entao a implementacao mobile ficou preparada conceitualmente enquanto a experiencia mobile de alta qualidade ja esta refletida no frontend responsivo em `apps/web`.
